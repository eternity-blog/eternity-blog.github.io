Markdown 相关语法

[TOC]

有关级数标题（1到6级标题）

直接选中文本，鼠标右键 -> 段落 -> X级标题

直接对应数量的#加上空格即可

使用快捷键： Ctrl + 上方键盘数字1到6

有关字体_加粗 （**）

直接选中文本，鼠标右键 -> B加粗

加粗语法格式：** **（中间为待加粗的文本）

使用快捷键： Ctrl + B

有关字体_倾斜（*）

直接选中文本，鼠标右键 -> I倾斜

倾斜语法格式： * *（中间为待倾斜的文本）

使用快捷键： Ctrl + I

有关字体_倾斜并加粗（***）

直接选中文本，鼠标右键 -> B加粗 -> I倾斜

语法格式：（用法类似于上面，但为左右均三个*）

使用快捷键： Ctrl + I 然后 Ctrl + B（顺序不分前后）

有关字体_高亮(==)

语法格式：== ==（中间为高亮文本）

==key==

有关字体_上标与下标(^and~)

语法格式：用 ^ 来包裹上标内容，用 ~ 来包裹下标内容

H2O和 H^2^O

有关字体_中划线（~~）

此功能不支持快捷键和右键菜单

语法格式： 在文本前后加上~~

有关字体_分割线（—）

语法： 三个-或者三个_或者三个 加回车即可*

下面是三个-的分割线效果

下面是三个_的分割线效果

下面是三个***的分割线效果

还可以右键 -> 插入 -> 水平分割线

有关引用（>）

右键 -> 引用按钮 创建引用框

语法格式：>加上空格加上文本

使用快捷键： Ctrl + shift + Q 快速生成一个引用框

points：如果在引用文本后加回车会默认继续处于引用状态，再敲一次回车即可退出引用

同时：引用框可以无限嵌套 嵌套几层便几个>

有关图片(! + [] + () )

points: 注意是英文状态下

[] 中即为图片的名称 默认为alt；

()中即为图片的位置 分为本地位置和网络图片

右键 -> 插入 -> 图片

使用快捷键： Ctrl + shift + I

本地导入可以直接拖入即可

有关超链接( )

语法格式：[链接名] + (href)

如：超链接

有关列表（- ）

右键菜单 选择 无序列表或者有序列表

语法格式：

无序列表：-或者+或者*（+空格）项目一

有序列表：1.（+空格）项目一

快捷键： Ctrl + shift + [ 生成有序列表 Ctrl + shift + ] 快速生成无序列表

points：在列表中创建子列表：在子列表前按下Tab键

有关表格（|单元格|）

建议直接右键插入

有关代码（`）

代码块语法格式： ‘代码块’

points：可以理解为注释，不会被识别为任何语言，即不会被渲染

有关代码块（````）

` 为 反单引号

代码块的语法格式：’’’’ + 语言名

也可用右键菜单
