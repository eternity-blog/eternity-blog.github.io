变量命名规则

  1. 只能由字母、数字、下划线组成
  2. 不能以数字开头
  3. 不能有空格

大小写敏感

不能以保留字命名

在 Python 3 中，可以用中文作为变量名，非 ASCII 标识符也是允许的了。但考虑到编码问题，一般还是采用英文命名法（下划线法、驼峰命名法）
