快速排序

又称为：分区交换排序

平均时间复杂度：O（nlogn）

最坏时间复杂度：O（n2）

核心思想：分治

  1. 选择分界点，可以是：a[l]，a[r]， a[(l + r + 1)/2]，等等

  2. 划分区间：将整个数组划分为小于等于分界点的区间和大于等于分界点的区间

    • 暴力做法

      可以另开两个数组a和b，分别存小于分界点的数和大于等于分界点的数，最后覆盖原数组

    • 优化做法

      双指针 i 和 j，从两头往中间走

  3. 分别递归左边和右边

快排属于分治算法，最怕的就是 n分成0和n，或 n分成n和0,这会造成无限划分

  • 若以 j 为划分时，x不能选 q[r]
  • 若以 i 为划分时，x不能选 q[l]

将整个区间一分为二，一半满足一半不满足某个条件，利用二分去寻找这个边界

有单调性一定可以二分，二分不一定要单调性

差分

与前缀和互为逆运算

已知：a1 a2 …… an 为前缀和数组，构造一个数组b，使得：b1 = a1 ，b2 = a2 - a1 …… bn = an - an-1

则称b数组为a数组的差分。

an = b1 + b2 + …… + bn O(n)：b -> a

用处：给定一个区间 [l，r]，使得 ai (i ∈ [l，r])，都有：ai = ai + C

如果直接遍历，时间复杂度为O(n)，

对其差分数组处理的话，时间复杂度为O(1)：只需要在b数组中修改两个数：bl = bl + C，br+1 = br+1 - C

  1. 先把第k位移到最后一位：n >> k

离散化

离散化的本质：映射，将间隔很大的点，映射到相邻的数组元素中，以减少对空间的需求，也减少计算量。

离散化：把无限空间中有限的个体映射到有限的空间中去，以此提高算法的时空效率。通俗的说，离散化是在不改变数据相对大小的条件下，对数据进行相应的缩小。

适用范围

数组中元素值域很大，但个数不是很多。

比如将a[] = [1,3,100,2000,500000]映射到 [0,1,2,3,4]，这个过程就叫离散化（将值Value映射到下标Index）
